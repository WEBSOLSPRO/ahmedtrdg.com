# Ahmed Trading & Services - Website Redesign Project

📌 Project Overview
I developed this comprehensive static website for **Ahmed Trading and Services**, a heavy machinery spare parts business based in Salalah, Oman. The project involved transforming a basic one-page WordPress site into a fully responsive, multi-page HTML/CSS/JS architecture tailored to showcase their extensive industrial inventory and improve user navigation.

#🔗 Live Links
* Client Domain (Final Delivered WordPress): https://ahmedtrdg.com/
* Static Code Preview (Vercel Deployment): https://ahmedtrdg.vercel.app/

🛠️ Built With
* HTML5: Semantic and accessible page structure.
* CSS3 (Vanilla): Custom, responsive design system utilizing CSS Grid and Flexbox, adhering to the client's strict Yellow/Black brand guidelines.
* JavaScript: Interactive UI elements.
* Responsive Design: Mobile-first approach ensuring perfect layouts across all devices.

✨ Key Features
* Multi-Page Architecture: Created distinct pages for Home, Products, About Us, and Contact, moving away from a single-page layout.
* Categorized Product Grid: Built an interactive 9-card category system displaying specific machinery parts (Filtration Systems, Undercarriage Parts, Hydraulic Breakers, etc.), significantly improving UX.
* Modern Header/Navigation: Implemented a sticky, responsive two-line header cleanly separating the company name from site navigation.
* SEO Optimized: Structured semantic HTML elements (h1-h6 formatting) and optimized image rendering for fast load times.
* Integrated Contact Section: Responsive inquiry form and an embedded interactive Google Map pinpointing the client's headquarters in Oman.


*This repository represents the static hard-coded version of the website created specifically for demonstration, source code review, and version control.*
